#define ADC_MAX_CHANNEL   3

To calculate this value you can use function: long map(long x, long in_min, long in_max, long out_min, long out_max){ return (x-in_min)*(out_max-out_min)/(in_max - in_min) + out_min; } val = map(1,0,3.3,0,1023). here (0-3.3V range mapped to 0-1023, and ADC value of 1V will return 310.)
